# wavoperatepy

wave,pyaudio,librosaを使って波形を見たり、AD(DA)変換したり、特徴量抽出したり

kerasを用いて、3秒の声を3分類して3種類の設定した音楽を流すGUIアプリ(PyQt)を作成中
